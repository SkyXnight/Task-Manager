# Task Manager

A simple and modern task management application built with Next.js.

## 🚀 Features

- Create, edit and delete tasks
- Mark tasks as completed
- Due date tracking
- Task categories / tags
- Responsive design
- User settings (theme, preferences)

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Prisma
- Tailwind CSS
- PostgreSQL

## 📦 Installation

```bash
git clone https://github.com/SkyXnight/task-manager.git
cd task-manager
npm install
npm run dev
